Pair with Target Sum Problem

Problem

Problem Description: Pair with Target Sum

Given an array of numbers sorted in ascending order and a target sum, find a pair in the array whose sum is equal to the given target.

Write a function to return the indices of the two numbers (i.e. the pair) such that they add up to the given target.

Example 1:

Input: [1, 2, 3, 4, 6], target=6

Output: [1, 3]

Explanation: The numbers at index 1 and 3 add up to 6: 2+4=6

Example 2:

Input: [2, 5, 9, 11], target=11

Output: [0, 2]

Explanation: The numbers at index 0 and 2 add up to 11: 2+9=11

Understanding

This problem asks us to find two numbers in a sorted array that add up to a specified target value, and return their indices.

The key elements of this problem are:

  • Sorted Array: The input array is already sorted in ascending order, which we can take advantage of.
  • Pair Sum: We need to find exactly two numbers whose sum equals the target.
  • Return Indices: The output should be the indices of these two numbers, not the numbers themselves.

For example, in the array [1, 2, 3, 4, 6] with target 6, the pair 2 and 4 sum to 6, and they're located at indices 1 and 3 respectively.

Since the array is sorted, if we pick two numbers and their sum is greater than the target, we need to consider smaller numbers. Conversely, if their sum is less than the target, we need to consider larger numbers.

Approach

Since the array is sorted, we can use the Two Pointers approach, which allows us to solve this problem efficiently:

Key Insight: By maintaining two pointers at opposite ends of the array and moving them based on the comparison of the current sum with the target, we can efficiently find the pair without checking all possible pairs.

The Two Pointers approach works as follows:

  1. Initialize two pointers, one at the start of the array (left) and one at the end (right).
  2. Calculate the sum of the elements at the left and right pointers.
  3. If the sum equals the target, we've found our pair.
  4. If the sum is greater than the target, decrement the right pointer to consider a smaller element.
  5. If the sum is less than the target, increment the left pointer to consider a larger element.
  6. Continue this process until either a pair is found or the pointers meet.

Visualization of the Two Pointers Approach:

Array: [1, 2, 3, 4, 6], Target: 6

Step 1: left = 0, right = 4
       [1, 2, 3, 4, 6]
        ^        ^
       left     right
       Sum = 1 + 6 = 7 > 6, so move right pointer left

Step 2: left = 0, right = 3
       [1, 2, 3, 4, 6]
        ^     ^
       left  right
       Sum = 1 + 4 = 5 < 6, so move left pointer right

Step 3: left = 1, right = 3
       [1, 2, 3, 4, 6]
           ^  ^
          left right
       Sum = 2 + 4 = 6 == 6, we found the pair!

Data Structure Choice:

We don't need any additional data structures for this approach, as we're just manipulating pointers on the input array. This gives us O(1) space complexity, which is optimal.

Time and Space Complexity:

  • Time Complexity: O(n) where n is the length of the array. In the worst case, we might need to traverse the entire array once.
  • Space Complexity: O(1) as we only use two pointers regardless of the input size.

Alternative Approach:

Another approach would be to use a hash table (dictionary) to store elements we've seen so far and check if the complement (target - current element) exists in the hash table. However, this would use O(n) space and doesn't take advantage of the fact that the array is already sorted. The Two Pointers approach is more efficient for this specific problem.

Steps

  1. Initialize two pointers:
    • Set the left pointer to the beginning of the array (index 0).
    • Set the right pointer to the end of the array (index length-1).
  2. Enter a loop that continues as long as left < right:
    • Calculate the current sum: nums[left] + nums[right].
  3. Compare the current sum with the target:
    • If sum > target: Decrement the right pointer to reduce the sum.
    • If sum < target: Increment the left pointer to increase the sum.
    • If sum == target: Return the current indices [left, right] as the answer.
  4. Return [-1, -1] if no pair is found:
    • If we exit the loop without finding a pair, return [-1, -1] to indicate failure.

Code

def pairWithTargetSum(nums, target):
    # Initialize two pointers at the beginning and end of the array
    left, right = 0, len(nums) - 1
    
    # Continue until the pointers meet
    while left < right:
        # Calculate current sum
        current_sum = nums[left] + nums[right]
        
        # If sum equals target, we found our pair
        if current_sum == target:
            return [left, right]
        
        # If sum is too large, decrement right pointer to reduce sum
        elif current_sum > target:
            right -= 1
        
        # If sum is too small, increment left pointer to increase sum
        else:  # current_sum < target
            left += 1
    
    # If no pair is found, return [-1, -1]
    return [-1, -1]

Example Run:

# Example input
nums = [1, 2, 3, 4, 6]
target = 6

# Call our function
pairWithTargetSum(nums, target)

Output:

[1, 3]

Detailed Trace:

  1. Initialize: left = 0, right = 4
    • Array at this point: [1, 2, 3, 4, 6]
    • Current sum: nums[0] + nums[4] = 1 + 6 = 7
    • 7 > 6, so decrement right: right = 3
  2. Next iteration: left = 0, right = 3
    • Current sum: nums[0] + nums[3] = 1 + 4 = 5
    • 5 < 6, so increment left: left = 1
  3. Next iteration: left = 1, right = 3
    • Current sum: nums[1] + nums[3] = 2 + 4 = 6
    • 6 == 6, we found our pair!
    • Return [1, 3]

Second Example:

# Example input
nums = [2, 5, 9, 11]
target = 11

# Call our function
pairWithTargetSum(nums, target)

Output:

[0, 2]

Explanation of Second Example:

In the array [2, 5, 9, 11], the function checks:

  • First sum: 2 + 11 = 13 > 11, so right pointer moves: right = 2
  • Next sum: 2 + 9 = 11 == 11, which matches our target
  • Return [0, 2] as these are the indices of 2 and 9
